# mighty-simple-cms
CMS for a 3-ish hour dev test

**Quick how-to guide:**

1) Clone repo
2) cd into the root of the project
3) run mpm i
4) run npm run dev
5) visit localhost:3000


---
A few comments:
- Delete end point is implemented, but add (post) and update (put) are not
- No login functionality for now
